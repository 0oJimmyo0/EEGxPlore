# About this fork

This repository extends [**CBraMod**](https://github.com/wjq-learning/CBraMod) (MIT license, see `LICENSE`) with local research changes, including:

- `attnres_variant` encoder options and related modules (e.g. `models/attn_res.py`)
- Updates to `finetune_main.py`, `finetune_trainer.py`, and `models/model_for_faced.py`
- Example cluster script: `scripts/submit_train.slurm`

Upstream paper and citation: see `README.md`. When publishing this fork, cite the original CBraMod work.
